S-BUZZ: A Coding Project

Live life in the fast lane. A javascript coding exercise using various functions and loops. Write a program that displays a range of numbers and replace those numbers with a user inputed term should they be a multiple of a user inputed number.

Instructions

Observe the user interface and notice the pairs of user input fields labeled "Term x" & "Multiple x." Also notice the "Start" & "End" labeled user input fields. Set the range of numbers by entering a start and end value in the corresponding fields. Enter a term and multiple per term-multiple pair fields. You may add term-multiple pair fields if you desire. When done, hit "S-Buzz" to display the results.
